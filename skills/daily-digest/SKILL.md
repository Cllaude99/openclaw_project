# 일일 종합 브리핑 스킬

## 사용 시점

- 크론 작업에 의한 자동 트리거 (아침 07:00, 점심 12:30, 장 마감 18:00)
- 사용자가 "브리핑", "요약", "종합" 등을 요청할 때

## 브리핑 유형

### 1. 아침 브리핑 (07:00 KST)
전체 종합: 날씨 + 금융 + 코인 + 뉴스

### 2. 점심 뉴스 (12:30 KST)
IT 뉴스 + 국내 주요 뉴스 중심

### 3. 장 마감 요약 (18:00 KST, 평일)
금융 시장 + 코인 시세 변동

## 실행 절차

다음 순서로 데이터를 수집합니다:

### Step 1: 날씨 데이터 수집
```bash
bash skills/weather-kr/scripts/fetch_weather.sh
```

### Step 2: 환율 데이터 수집
```bash
bash skills/finance-kr/scripts/fetch_exchange.sh
```

### Step 3: 주가지수 데이터 수집
```bash
bash skills/finance-kr/scripts/fetch_stocks.sh
```

### Step 4: 암호화폐 데이터 수집
```bash
bash skills/crypto-kr/scripts/fetch_crypto.sh
```

### Step 5: 기술 뉴스 수집
```bash
bash skills/tech-news/scripts/fetch_hn.sh
```

### Step 6: 일반 뉴스 수집
```bash
bash skills/news-summary/scripts/fetch_news.sh
```

## 메시지 제한 처리

- Telegram 메시지 제한: 4096자
- `textChunkLimit: 4000`으로 설정되어 자동 분할
- 분할 시 각 메시지가 의미 있는 단위로 나뉘도록 섹션 단위로 구성
- 아침 브리핑이 길면: 1메시지(날씨+금융+코인) + 2메시지(뉴스)로 분할

## 오류 처리

- 특정 API 실패 시 해당 섹션만 "현재 데이터를 가져올 수 없습니다"로 표시
- 나머지 섹션은 정상 출력
- 전체 API 실패 시 "브리핑 생성에 실패했습니다. 잠시 후 다시 시도해주세요."
